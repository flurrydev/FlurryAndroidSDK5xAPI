Detailed Description

Listener for FlurryAgent.

Author
2009-2014 Flurry, Inc. All Rights Reserved

Flurry will do a one time authentication of the user and then post the shared content to Tumblr.

* PhotoPost postImage = new PhotoPost("Your App’s photo to post");
postImage.setCaption("caption");
postImage.setAndroidDeeplink("Your App’s Android Deep Link to Content");
postImage.setIOSDeepLink("Your App’s iOS Deep Link to Content");
postImage.setWebLink("Your App’s Web Link to Content");
postImage.setPostListener(fPostListener);
TumblrShare.post(mContext, postImage);

static Bitmap getTumblrImage ( )
static

Api to get the default Tumblr logo. The Bitmap returned has a default size of 35dp * 35dp. If you are planning to use different sizes please scale the bitmap using below api.

Bitmap.createScaledBitmap(TumblrShare.getTumblrImage(), widthInPixel, heightInPixel);

Returns
Bitmap of size 35dp * 35dp
