Detailed Description

Listener for Native ads.

Author
2009-2014 Flurry, Inc. All Rights Reserved

The listener for Native ads. A listener is set of callback methods that allow a developer to manage and take actions within different phases of ad display.

See Also
Flurry Publisher Support Site for more information
